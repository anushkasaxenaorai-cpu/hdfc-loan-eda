# 📊 HDFC Loan Dataset - Exploratory Data Analysis (EDA)

This project is a beginner-level Data Analysis project created during a workshop.  
It focuses on understanding loan data using Python libraries.

---

## 📌 Objective
To analyze the HDFC loan dataset and extract meaningful insights such as:
- Missing values
- Dataset structure
- Data quality issues

---

## 🛠️ Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📈 Key Analysis Performed
- Displayed dataset head and tail
- Checked dataset shape (rows & columns)
- Identified missing values
- Calculated missing value percentages
- Visualized missing data using heatmap

---

## 📁 Files in Repository
- `main.py` → Python script for data analysis
- `README.md` → Project documentation

---

## 🎓 Learning Outcome
This project helped in understanding basic Exploratory Data Analysis (EDA) and data cleaning techniques.
